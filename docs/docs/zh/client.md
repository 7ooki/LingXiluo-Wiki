# 如何启动游戏
## 1. 下载客户端压缩包
- 使用浏览器(Chrome/Edge)访问 下载链接 并下载
  - 123网盘: [下载链接](https://example.com/download)
  - 蓝奏云: [下载链接](https://example.com/download)
  - Cloudflare: [下载链接](https://example.com/download)
## 2. 解压客户端

- 使用 解压软件 解压 压缩包(例如: WinRAR)
- 进入解压后的文件夹

## 3. 启动客户端

- 运行 `PCL2.exe`
- 登录 你的 正版账号(无正版可用离线登录)
- 启动游戏
